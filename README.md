Disang Hostel Website

A responsive web application developed to represent Disang Hostel (IIT Guwahati), providing centralized access to hostel-related information including administration details, infrastructure overview, and contact interface.

Features
Responsive and structured user interface
Hostel administration (HMC) information display
Informational sections covering hostel facilities and overview
Contact interface for user queries
Simple navigation with fast load times
Tech Stack
Frontend: HTML, CSS, JavaScript
Deployment: GitHub Pages

Overview

This project is a static web application aimed at digitizing hostel-level information and improving accessibility for students and visitors. It consolidates administrative and general hostel information into a single interface, reducing reliance on fragmented or informal communication channels.

Installation
Clone the repository
git clone https://github.com/sb-2006/Disang-hostel-website.git
Navigate to the project directory
cd Disang-hostel-website
Open index.html in a web browser
